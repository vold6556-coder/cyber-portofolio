# Finding 03 â€“ Directory Listing Aktif

## Deskripsi
Direktori aplikasi dapat diakses secara anonymous dan menampilkan halaman "Index of /" tanpa autentikasi.

## Dampak
- Kebocoran struktur internal aplikasi
- Mempermudah reconnaissance lanjutan
- Membantu pencarian CVE berdasarkan dependency

## CVSS v3.1
**Score:** 5.3 (Medium)

## Proof of Concept (Disanitasi)
- Directory listing ditemukan menggunakan Google Dork
- Screenshot disanitasi tersedia di appendix
![PoC Directory Listing](../../appendix/Screenshots_Sanitized/Security_Misconfiguration/03-poc_directory-listing.png)


## Mitigasi
- Nonaktifkan directory listing
- Terapkan Role-Based Access Control (RBAC)
- Audit repository publik
- Hardening konfigurasi artifactory
